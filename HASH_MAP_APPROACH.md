# Hash Map Approach - Optimal Solution

## Overview
This is an optimized O(n) solution using a hash map to find two sum.

## Algorithm
1. Create a hash map
2. Iterate through array once
3. For each number, check if (target - number) exists
4. If yes, return indices
5. If no, store current number in hash map

## Time Complexity: O(n)
## Space Complexity: O(n)
