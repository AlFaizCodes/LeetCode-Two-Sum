# Input and Output Examples

## Example 1

**Input:**
```
nums = [2, 7, 11, 15]
target = 9
```

**Output:**
```
[0, 1]
```

**Explanation:**
The element at index 0 is 2 and the element at index 1 is 7.
Because nums[0] + nums[1] == 9, we return [0, 1].
